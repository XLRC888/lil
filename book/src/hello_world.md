# Hello, World

Create a file called `hello.lil`:

```lil
print "hello, world"
```

Run it:

```bash
lil hello.lil
```

The `print` statement outputs text to stdout and appends a newline. You can print numbers and strings:

```lil
print 42
print "hello"
```

You can print multiple values by separating them with commas:

```lil
print "the answer is", 42
```

This prints: `the answer is 42`
